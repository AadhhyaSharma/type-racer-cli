## What is this?

A terminal typing game I built because I wanted to track my WPM without opening a browser. Type the prompt, see your live WPM and accuracy, beat your high score.

Scores are saved locally in `scores.json` so your leaderboard persists between sessions.

---

## Usage

```bash
python racer.py             # random prompt, start typing
python racer.py --hard      # longer, harder prompts
python racer.py --scores    # view your leaderboard
python racer.py --reset     # clear all scores
```
